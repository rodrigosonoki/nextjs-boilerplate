This is a [Next.js](https://nextjs.org/) boilerplate template.

## Details

#### NextJS configurations

- Typescript
- ESLint
- Tailwind CSS
- `src/` directory
- App Router
- Import alias as `@/*`

#### CSS Reset

```css
* {
  margin: 0;
  padding: 0;
  list-style: none;
  vertical-align: baseline;
}
```

## Installation

First, install the dependencies:

```bash
npm install

# or

yarn install
```

and then run the development server:

```bash
npm run  dev

# or

yarn dev
```
